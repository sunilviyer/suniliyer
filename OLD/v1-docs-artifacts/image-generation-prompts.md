### Step 2: Generation Settings

**Midjourney Settings**:
- Version: `--v 6`
- Style: `--style raw` (less stylization, more literal)
- Aspect ratio: `--ar 3:2` (concept cards), `--ar 16:9` (heroes)
- Quality: Default (faster generation)

**DALL-E 3 Settings**:
- Size: 1792x1024 (then crop/resize to exact specs)
- Style: Natural (more realistic)
- Quality: HD

### Step 3: Post-Processing

1. **Resize**: Ensure exact dimensions (1920x1280, 1920x1080, 1200x800)
2. **Optimize**: Export as JPG (quality 85) + WebP (quality 80)
3. **Naming**: Use exact filenames from this document
4. **Organize**: Save to correct `/public/images/learn/{category}/` folder
