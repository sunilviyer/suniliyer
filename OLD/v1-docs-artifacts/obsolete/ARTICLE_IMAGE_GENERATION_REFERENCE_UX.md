# Article Image Generation Reference - UX Design System

**Date:** December 18, 2025  
**Purpose:** Complete reference for generating images for all 167 articles using the AIDefence UX Design System  
**Usage:** Use the prompts with DALL-E, Midjourney, or Stable Diffusion to create article header images  
**Design System:** Based on earth-tone palette with category-coded visual motifs

---

## Design System Overview

### Earth-Tone Color Palette

| Color Name | Hex Code | Usage |
|------------|----------|-------|
| **Charcoal Brown** | `#333d29` | Dark mode background, light mode primary text |
| **Moss Green** | `#414833` | Dark mode surface elements |
| **Fern** | `#656d4a` | Dark mode elevated surfaces |
| **Dry Sage** | `#a4ac86` | Category color (Legal Frameworks) |
| **Khaki Beige** | `#c2c5aa` | Category color (Governance) |
| **Tan** | `#b6ad90` | Category color (Dev Lifecycle) |
| **Camel** | `#a68a64` | Category color (Risks & Principles) |
| **Toffee Brown** | `#936639` | Category color (AI Fundamentals) |
| **Saddle Brown** | `#7f4f24` | Category color (AI Laws) |
| **Dark Walnut** | `#582f0e` | Category color (Risk Frameworks) |
| **Beige Ivory** | `#F5F5DC` | Light mode background |

### Category Visual Motifs

- **AI Fundamentals:** Neural network nodes, brain synapses, learning pathways
- **AI Risks & Principles:** Balanced scales, warning shields, ethical symbols
- **Legal Frameworks:** Legal documents, scales of justice, regulatory symbols
- **AI Laws:** Government buildings, legislative halls, compliance checkmarks
- **Risk Frameworks:** Shield matrices, safety nets, protective barriers
- **AI Development Lifecycle:** Circular processes, iterative cycles, pipeline flows
- **Governance Implementation:** Organizational charts, strategic roadmaps, building blocks
- **AI Auditing & Accountability:** Magnifying glasses, checklists, inspection tools
- **Industry-Specific Insights:** Sector icons, business contexts, application domains
- **AI Explainability:** Transparent boxes, light rays, clarity symbols

### Image Specifications

- **Master Size:** 1200x750px (16:10 aspect ratio)
- **Card Display:** 400x250px (scaled down)
- **Style:** Abstract geometric illustration, minimalist, professional
- **Color Scheme:** Earth tones with category-specific accent colors
- **Background:** Subtle gradients using category colors
- **Contrast:** High contrast for accessibility (7:1 ratio)

## Implementation Guidelines

### Image Generation Workflow

1. **Select Category Template:** Choose the appropriate category-based prompt template
2. **Customize for Article:** Add article-specific elements and context
3. **Generate Master Image:** Create 1200x750px version using the prompt
4. **Optimize for Web:** Create optimized versions for different display sizes
5. **Test Accessibility:** Ensure 7:1 contrast ratio for overlaid text

### File Naming Convention

```
[article-slug]-hero-image.png
[article-slug]-hero-image-400x250.png (card version)
[article-slug]-hero-image-800x500.png (medium version)
```

### CSS Integration

```css
.article-card-image {
  background: linear-gradient(135deg, var(--category-color) 0%, var(--category-color-light) 100%);
  border-radius: 8px 8px 0 0;
  aspect-ratio: 16/10;
}

.article-header-image {
  filter: blur(20px);
  opacity: 0.15;
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}
```

### Quality Checklist

- [ ] Earth-tone color palette maintained
- [ ] Category color prominently featured
- [ ] Minimalist, professional aesthetic
- [ ] High contrast for accessibility
- [ ] Geometric, abstract style
- [ ] 1200x750px master resolution
- [ ] Suitable for both light and dark themes
- [ ] No text or specific branding elements
- [ ] Scalable to smaller sizes (400x250px)
- [ ] Consistent with overall design system

## Usage Notes

- **Batch Generation:** Use category templates for efficient batch processing
- **Consistency:** Maintain visual consistency within each category
- **Accessibility:** Always test with overlaid text for readability
- **Performance:** Optimize images for web delivery
- **Fallbacks:** Provide category-colored gradients as fallbacks
- **Testing:** Preview images in both light and dark themes

This reference ensures all article images align with the AIDefence UX Design System while maintaining category-coded visual organization and earth-tone aesthetic consistency.
