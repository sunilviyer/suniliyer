## How to use this tracker

For each article, fill in the edits you want under its section. Use this format for each edit:

- **Location:** (section heading, paragraph, or a short quote of the current text)
- **Current:** what it says now
- **Change to:** what it should say
- **Reason / notes:** (optional)

Set the article's **Status** using the legend below.

### Status legend

| Status | Meaning |
|--------|---------|
| ⬜ Not reviewed | Haven't read this article yet |
| 🔍 Reviewing | Currently reading / identifying edits |
| ✏️ Edits logged | Edits identified and written below |
| ✅ No changes | Read, no edits needed |
| 🔧 Applied | Edits have been made to the file |
