## User Story

**I want** keyboard shortcuts
**So that** I can navigate efficiently

---

## Acceptance Criteria

- [ ] All shortcuts work
- [ ] Help modal accessible via `?`
- [ ] No conflicts with browser shortcuts

---

## Shortcuts

- `/` - Focus search
- `b` - Bookmark current card
- `n` - Open notes
- `↑/↓` - Navigate cards
- `Esc` - Close modals
- `?` - Show shortcuts help
