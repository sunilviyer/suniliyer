## Image Creation Guidelines

### Technical Specifications:
- **Format**: JPG (WebP for production after optimization)
- **Concept Cards**: 1920x1280 (3:2 ratio)
- **Hero Images**: 1920x1080 (16:9 ratio)
- **Example Cards**: 1200x800 (3:2 ratio)
- **Resource Cards**: 1200x800 (3:2 ratio)

### Style Guidelines:
1. **Color Palette**:
   - Primary: Moss Green (#333d29)
   - Accent: Brown Tan (#936639)
   - Secondary: Olive Green (#414833)
   - Neutral: Light grays for backgrounds

2. **Visual Style**:
   - Modern, professional
   - Slightly abstract/conceptual
   - Clean, minimal composition
   - High contrast for readability
   - Consistent visual language across paths

3. **Content**:
   - Include relevant icons/symbols
   - Use diagrams where appropriate
   - Incorporate logos (OECD, UNESCO, company logos) where mentioned
   - Avoid text-heavy images (text is in the cards themselves)

4. **Accessibility**:
   - Ensure images work in both light and dark themes
   - High contrast ratios
   - Avoid relying solely on color to convey meaning

## Image Fallbacks

If specific images aren't available, use placeholder pattern:
- Background: Subtle gradient using path-specific color
- Icon: Simple, relevant SVG icon
- Pattern: Geometric pattern in brand colors

Example fallback naming:
- `placeholder-history.jpg`
- `placeholder-terminology.jpg`
- etc.
