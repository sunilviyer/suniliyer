1 What is Machine Learning?

Machine learning enables computers to learn from data without explicit programming, using statistical patterns to make predictions and decisions.

Core Concept

Machine learning is a subset of artificial intelligence that focuses on building systems that learn from and make decisions based on data. Rather than following explicitly programmed rules, ML algorithms identify patterns in data and use those patterns to make predictions or decisions without human intervention.

Tom Mitchell's formal definition (1997) states: "A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E."

Key Insight

The power of machine learning lies in its ability to discover patterns that humans might not recognize, especially in high-dimensional data spaces with millions of variables.

Three Learning Paradigms

Paradigm Training Data Learning Mechanism Example Use Case
Supervised Learning Labeled examples Learn patterns mapping inputs to outputs Image classification, spam detection
Unsupervised Learning Unlabeled data Discover hidden patterns and structure Customer segmentation, anomaly detection
Reinforcement Learning Reward signals Learn through trial-and-error interaction Game playing, robotics control

Historical Context

The term "machine learning" was coined by Arthur Samuel in 1959 while working on a checkers-playing program at IBM. However, the theoretical foundations trace back to the 1940s with early work on neural networks and pattern recognition.

Governance Implications

  • Transparency: ML models can be "black boxes," making it difficult to explain why specific decisions were made
  • Bias Amplification: Models learn from historical data, potentially perpetuating or amplifying existing biases
  • Data Privacy: ML systems require large datasets, raising concerns about data collection and use
  • Accountability: When ML systems make mistakes, determining responsibility becomes complex

Regulation Watch

The EU AI Act (2024) classifies ML systems by risk level, with high-risk applications like hiring and credit scoring facing strict transparency and testing requirements.

2 Early Foundations (1950s-1960s)

The 1950s-1960s established the theoretical foundations with the perceptron, early neural networks, and pioneering work in pattern recognition.

The Perceptron (1957)

Frank Rosenblatt's perceptron was the first algorithm capable of learning from examples. This single-layer neural network could classify inputs into one of two categories by adjusting weights based on training errors.

Key Breakthrough

The perceptron learning algorithm demonstrated that machines could automatically learn to classify patterns without being explicitly programmed for every scenario.

Limitations and Setbacks

Marvin Minsky and Seymour Papert's 1969 book "Perceptrons" proved that single-layer perceptrons couldn't solve the XOR problem (a simple logical function). This limitation, combined with funding challenges, contributed to the first "AI Winter."
