Next Installation

Please follow the instructions in the video to see how you can install Next on your hosting:

  1. For local host -
    1. Open you command prompt
    2. npm install
    3. npm dev (will start the dev server at http://loaclhost:3000)
  2. Deploying to Netlify - https://youtu.be/M7TJeCk4iL8?si=7xXP229oNRChQykc


  1. For locally build -
    1. Open you command prompt
    2. npm build (will build locally and generate public directory)
    3. npm start (will start the locally build server at http://loaclhost:3000)
  2. Deploying to Vercel - https://youtu.be/M7TJeCk4iL8?si=7xXP229oNRChQykc
