🎨 Layer Generation Prompts (if using images instead of CSS)

Layer 1 - Purple Orb

Soft radial gradient orb, purple (#9333ea) to transparent, heavy gaussian blur (80px), transparent PNG, 800x800px, centered gradient

Layer 2 - Blue Gradient

Elliptical gradient, blue (#6366f1) to transparent, blur (100px), horizontal orientation, 1000x600px, transparent PNG

Layer 3 - Violet Sphere

Circular gradient orb, violet (#8b5cf6) to transparent, blur (90px), 600x600px, transparent PNG, slightly off-center gradient

Note: Can be pure CSS gradients for zero file size, or generated PNGs for more complex effects.

✅ Why Parallax Layers Work

  • Seamless: No breaks in reading flow - atmosphere is continuous
  • Performant: CSS-only solution = 60fps smooth scrolling
  • Lightweight: Tiny file sizes compared to photos
  • Depth: Parallax creates 3D perception without being gimmicky
  • Flexible: Easy to adjust colors, speeds, opacities per section
  • Modern: Trendy glassmorphism aesthetic that feels premium
